# HN Deck

A tweetdeck like interface to browse Hacker News

Live https://hndeck.sagunshrestha.com/

## Pre requisites

* Python 3.6+
* pip

## Install Requirments

    $ pip install -r requirements.txt
    
## Run App

Collect data, alternatively schedule fetch_data.py as cron job

    $ python fetch_data.py
    
Run server

    $ python app.py
