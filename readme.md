# Hacker MD

Top medicine-related stories from Hacker News.

www.hackermd.net

## Prerequisites

* Python 3.6+
* pip

## Install Requirements

    $ pip install -r requirements.txt
    
## Run App

Collect data or schedule fetch_data.py as a cron job

    $ python fetch_data.py
    
Run server

    $ python app.py
