# Hacker MD

Medicine-related stories from Hacker News.

Live https://www.hackermd.net

## Prerequisites

* Python 3.6+
* pip

## Install Requirments

    $ pip install -r requirements.txt
    
## Run App

Collect data, alternatively schedule fetch_data.py as cron job

    $ python fetch_data.py
    
Run server

    $ python app.py
